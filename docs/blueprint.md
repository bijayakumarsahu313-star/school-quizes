# **App Name**: QuizWhiz Academy

## Core Features:

- Teacher Dashboard: Centralized hub for teachers to manage classes, quizzes, and student progress.
- AI-Powered Quiz Generation: AI tool assists teachers in creating questions tailored to specific subjects, classes, and difficulty levels.
- Versatile Question Bank: Support for MCQs, True/False, Fill in the Blanks, Match the Following, and Image-based questions.
- Automated Grading: Instant auto-grading of quizzes with detailed performance reports.
- Student Zone: Clean interface with gamified quizzes, progress tracking, and personalized feedback.
- Curriculum Alignment: Support for CBSE, ICSE, State Boards, and custom syllabus to align with various educational standards.
- Analytics and Reporting: Comprehensive reports and analytics for teachers and schools to track student performance and identify areas for improvement.

## Style Guidelines:

- Primary color: A vibrant blue (#3498db) to convey trust and innovation.
- Background color: A light gray (#f0f2f5), almost white, to provide a clean and distraction-free environment.
- Accent color: An analogous light-green (#2ecc71), brighter than the primary and background, to highlight key interactive elements and CTAs.
- Body and headline font: 'PT Sans', a humanist sans-serif font.
- Use simple, clear icons to represent quiz sections, subjects, and actions. Maintain consistency in style and color.
- Design a responsive layout that adapts seamlessly to different screen sizes (desktops, tablets, and mobile devices).
- Incorporate subtle animations to provide feedback on user interactions and guide them through the quiz-taking process.
